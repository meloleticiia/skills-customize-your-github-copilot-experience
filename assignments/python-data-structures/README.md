# 📘 Assignment: Estruturas de Dados em Python

## 🎯 Objective

Aprender sobre as principais estruturas de dados em Python: listas, dicionários, sets e tuplas. Os alunos devem criar programas que demonstrem o uso dessas estruturas para resolver problemas simples.

## 📝 Tasks

### 🛠️	Trabalhando com Listas e Dicionários

#### Description
Crie um programa que gerencie uma lista de alunos e suas notas. Use uma lista para armazenar os nomes dos alunos e um dicionário para associar cada nome a uma lista de notas.

#### Requirements
Completed program should:

- Permitir adicionar alunos e suas notas
- Calcular a média das notas de cada aluno
- Exibir a lista de alunos com suas médias


### 🛠️	Usando Sets e Tuplas

#### Description
Crie um programa que trabalhe com conjuntos (sets) para encontrar elementos únicos e tuplas para dados imutáveis, como coordenadas.

#### Requirements
Completed program should:

- Usar um set para remover duplicatas de uma lista de números
- Usar tuplas para representar pontos em um plano cartesiano
- Calcular a distância entre dois pontos representados por tuplas
